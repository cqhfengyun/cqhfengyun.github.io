      • 1.2 更新说明

      1. 新版本使用了象棋的新规则，我称其为“升变”。规则和普通的规则有些不同。升变只需点击同一个棋子。
        相关链接：仕相可以升变成兵——中国象棋规则改革的探索
        以下是升变规则介绍：

        • 一、仕(士)相(象)只能在底线升变；
        • 二、仕(士)相(象)升变成兵(卒)需要花一步棋，变成的兵(卒)占据原来仕(士)相(象)的位置，下一步棋才能走；
        • 三、升变是不可逆的，即兵(卒)不能变回仕(士)相(象)；
        • 四、升变后的兵(卒)同其他兵(卒)的走法一样，过河前每步只能前行一格，过河后才能横走。

      3. 优化了客户端。单机游戏下没有人机，可以随便选择任意一方阵营的棋子，会显示可以移动的位置。不能不合规则地移动。联机游戏下需要按顺序下棋，但是程序不会判断胜负，自己看着办吧。

      4. 优化了服务器，低版本的客户端将无法连接服务器。但还是只能由ID1和ID2的玩家进行游戏，其他ID的玩家无法进行操作。

      5. 请无视客户端里的帮助，我一直没有更新。

      1.1 更新说明

      1. 每次选择棋子或移动棋子都会显示轨迹，虽然有点丑，但勉强够用。

      2. 优化了服务器，如果有人退出游戏将重新统计人数，所以当前人数是所有与服务器保持连接的客户端数量。而且空的的ID将会按顺序分配给新玩家。但是仍然只能由ID1和2的玩家进行游戏，所以如果你想进行游戏但ID大于2，而且总人数小于3，你可以断开与服务器的连接，再重连，服务器将会按顺序给你分配空的ID。

      3. 禁用了重启服务器的功能，因为已经没必要了。旧版本发送重启服务器的消息会被服务器忽略。

      4. 更新说明比发布说明更新，所以请以更新说明为准！

      1.0 发布说明

          历经艰辛，我的象棋发布测试版终于做好了，不过只能在Windows上运行，增加了联机对战的功能。理论上只要能联网，就能进行联机对战，当然还是有很多问题的。
          如果是进行单机游戏的话应该没什么问题，只是没有人机，只能自己瞎点。
          如果想体验联机模式的话，需要注意几个问题。我先介绍一下内部原理，让你们更好理解。我的象棋服务器挂在云服务器上，一直不关。象棋客户端点击联机游戏后，就会和服务器连接，服务器数据里的玩家数量就会增加一个，按顺序给每个玩家分配ID。但是象棋服务器只允许ID1和ID2玩家进行游戏,而且服务器不会清除断开连接的玩家信息，只要分配了一个ID就不会再给其他玩家分配这个ID了。所以我增加了一个服务器重启的功能，重启后会清除所有数据，ID也是重新分配。
          所以划重点，如果想要体验联机模式，请按以下步骤。一般不会有其他人使用，所以只能自己开两个客户端。第一次进入联机游戏大厅如果ID不是1请按R键重启服务器，然后返回主菜单，就可以正常使用了。只要ID1和2的客户端都准备游戏就会开始游戏。

      快捷键
          游戏时按鼠标中间的按键可以旋转棋盘。按回车键或者Esc键打开游戏菜单。联机游戏大厅按R键重启服务器，重启服务器以后记得先回到主菜单。

      常见问题

      1. 连接不上服务器。服务器故障或者你的网不好，或者服务器的网不好或者服务器比较卡(服务器配置比较差，体谅一下)

      2. 不管怎么点总人数都是0。原因同上。

      3. ID大于3。请按R键重启服务器，然后返回主菜单，就可以正常使用。

      4. 莫名其妙断开连接。有可能是其他人在重启服务器，不过应该没多少人会用，这种情况概率较小。

      5. 提示图片加载失败。和exe一起的有个image文件夹，使用时exe和image文件夹要在同一个目录下。
