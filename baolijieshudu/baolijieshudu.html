      • 发布说明

          这个软件有点简陋，但是能解出一些简单的题目，不过使用方法有点麻烦。小提示，如果输入完整的数独，就可以生成无数个新的数独题目，不过输入的不是正确的完整数独会有bug。

      使用方法
          将数独题目在任何能打字的地方按下面方式排列，空的地方填0。然后点确定。你可以把示例复制粘贴尝试一下。
      示例：
          0 3 0 0 0 7 0 0 4
          6 0 2 0 4 1 0 0 0
          0 5 0 0 3 0 9 6 7
          0 4 0 0 0 3 0 0 6
          0 8 7 0 0 0 3 5 0
          9 0 0 7 0 0 0 2 0
          7 1 8 0 2 0 0 4 0
          0 0 0 1 6 0 8 0 9
          4 0 0 5 0 0 0 3 0
